# ML Pipeline Project

Проект для классификации вин с использованием DVC для управления данными и MLflow для трекинга экспериментов.

## Цель проекта

Создание воспроизводимого MLOps пайплайна для:
- Управления версиями данных и моделей с помощью DVC
- Трекинга экспериментов и метрик через MLflow
- Автоматизации подготовки данных и обучения моделей
- Обеспечения полной воспроизводимости экспериментов

## Как запустить

```bash
# 1. Клонирование репозитория
git clone https://github.com/kvsemn/ml_prod.git
cd ml_prod

# 2. Установка зависимостей
pip install -r requirements.txt

# 3. Запуск пайплайна
dvc repro

# 4. Загрузка данных через DVC
dvc pull

# 5. Запуск MLflow UI (в отдельном терминале)
mlflow ui --backend-store-uri sqlite:///mlflow.db --port 5000
